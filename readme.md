# ANONMSG

Anonmsg is a website with a anonymous message system made in html, js, css nodeJS, socketIO and mySQL

## Installation

```bash

git clone https://github.com/unwxnted/anonmsg.git

cd anonmsg

npm i

```

## Usage

```
cd backend

node server.js

```
